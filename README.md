**Library Management System**
A full-stack Library Management System built with Spring Boot backend and React frontend. This application provides comprehensive management of books, members, loans, and users with role-based access control.
Core Functionality

•	Book Management: Add, edit, delete, and search books with availability tracking

•	Member Management: Manage library members with membership IDs

•	Loan Management: Track book loans, returns, and renewals

•	User Management: Role-based user accounts (Admin, Librarian, User)

•	Dashboard: Overview of library statistics and activities

Security & Authentication

•	JWT-based authentication

•	Role-based authorization (ROLE_ADMIN, ROLE_LIBRARIAN, ROLE_USER)

•	Password encryption with BCrypt

•	CORS configuration for frontend-backend communication

Technical Features

•	RESTful API design

•	PostgreSQL database integration

•	Responsive React frontend

•	Real-time search and filtering

•	Automated data initialization with sample data

Backend

•	Java 17 with Spring Boot 3.2.0

•	Spring Security with JWT authentication

•	Spring Data JPA for database operations

•	PostgreSQL as primary database

•	Maven for dependency management

Frontend

•	React 18 with functional components and hooks

•	React Router for navigation

•	Axios for API communication

•	CSS3 for styling with responsive design
 (v3.6 or later)
